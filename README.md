# 延伸阅读
**为什么叫做Kruth-Eye?**

Kruth是英文单词Truth的变体，将首字母改为K是因为本项目使用Kotlin实现。
并且Kruth-Eye和荷鲁斯之眼谐音。在游戏王决斗怪兽的最后一集，荷鲁斯之眼见证了
武藤游戏和阿图姆之间的最后一次决斗，鉴定了阿图姆的法老王身份，打开了冥界之门。
因此本项目命名为Kruth-Eye。
